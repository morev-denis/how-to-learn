# Проект "Научиться учиться"

## Описание проекта

Данный проект является учебным, созданным при прхождение курса Яндекс Практикум.
Проект призван закрепить следующие навыки (hard skills):
  * Разметка страницы семантическими тегамиж;
  * Позиционирование элеметнов;
  * Flexbox-вёрстка;
  * Вёрстка с использованием БЭМ методологии;
  * Применение переходов и анимации;


В данном проекте используются следующие технологии и методологии:
  * HTML5;
  * CSS3;
  * БЭМ;
  * Файловая структура БЭМ nested.

## Инструкция по просмотру

Для корректного просмотра необходимо использовать один из следующих брауезров:
  * Firefxox (версия 28 и выше);
  * Google Chrome (версия 21 и выше);
  * Microsoft Edge (версия 12 и выше).

## Планы

В дальнейшем планируется улучшить кроссбраузерность проекта за счёт добавления вендорных префиксов.
