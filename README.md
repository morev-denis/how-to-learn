# Проект "Научиться учиться"

https://morev-denis.github.io/how-to-learn/

## Описание проекта

Данный проект является учебным, созданным при прохождение курса Яндекс Практикум.
Проект призван закрепить следующие навыки (hard skills):

- Разметка страницы семантическими тегами;
- Позиционирование элементов;
- Flexbox-вёрстка;
- Вёрстка с использованием БЭМ методологии;
- Применение переходов и анимации;

В данном проекте используются следующие технологии и методологии:

- HTML5;
- CSS3;
- БЭМ;
- Файловая структура БЭМ nested;
- Webpack.

## Инструкция по просмотру

Для корректного просмотра необходимо использовать один из следующих браузеров:

- Google Chrome (версия 64 и выше);
- Mozilla Firefxox (версия 50 и выше);
- Safari (версия 11.1 и выше);
- Microsoft Edge (версия 17 и выше);
- Internet Explorer (версия 11 и выше).
